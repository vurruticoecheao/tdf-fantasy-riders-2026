# Tour de France Fantasy Riders 2026

Pagina estatica con la base de riders del Tour de France Fantasy 2026, incluyendo dorsales oficiales, filtros, ordenamiento y descarga del CSV filtrado.

## Datos

- Base de riders: Fantasy TdF Helper.
- Dorsales: lista oficial de starters en `letour.fr/en/riders`.
- Archivo servido por la pagina: `assets/riders_tdf_2026_with_jersey_numbers.csv`.
